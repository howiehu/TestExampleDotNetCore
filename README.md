# TestExample（.Net Core 版）

本练习用于训练和学习在 .Net Core 技术栈下，使用流行工具进行测试编写，已经包含的工具为：

- 测试框架：[xUnit.net](https://xunit.github.io/)
- 流式断言库：[Fluent Assertions](https://fluentassertions.com)
- 测试替身工具：[Moq](https://github.com/moq/moq4)

## 练习要求

使用以上测试工具（本练习不需要使用测试替身工具），在 `ExpenseServiceTest` 测试文件中针对 `ExpenseService` 实现类编写完整的单元测试。

练习点：

- 如何编写单元测试？
- 需要写几个测试？分别都测什么？
- 如何测试异常处理？
